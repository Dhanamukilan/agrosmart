# AgroSmart 🌾

AgroSmart is a comprehensive platform designed to empower farmers with modern agricultural insights, real-time advice, and an AI-powered assistant.

## Features ✨

- **Crop Advisory:** Detailed guides for various crops including soil types, seasons, fertilizers, and expected yields.
- **AI Assistant:** An intelligent chatbot to answer farming queries and help manage crops.
- **Weather Insights:** Live weather updates to plan agricultural activities.
- **Modern UI:** Built with HTML, CSS, and Bootstrap for a premium, responsive experience.

## Setup 🚀

To get the AI Chatbot working, you need to set up your OpenAI API key:

1. Copy `config.example.js` to `config.js`:
   ```bash
   cp config.example.js config.js
   ```
2. Open `config.js` and replace `'YOUR_API_KEY_HERE'` with your actual [OpenAI API Key](https://platform.openai.com/api-keys).
3. Open `index.html` in your browser to start using AgroSmart!

## Technologies Used 🛠️

- **Frontend:** HTML5, Vanilla CSS, Bootstrap 5
- **AI Integration:** OpenAI API
- **Icons:** Font Awesome
